Welcome to MyServer Backend API

This backend powers a role-based program management platform. All endpoints are secured and require authentication.

Authentication

  • POST /auth/login – Login with username & password. Returns accessToken and sets refreshToken cookie.
  • GET /auth/refresh – Use refresh token cookie to obtain a new access token.
  • POST /auth/logout – Logout and clear refresh token.

After login, include the accessToken in request headers: Authorization: Bearer <accessToken>

Available Dashboard Endpoints

  • GET /dashboard/activists – List all activists (Admin only)
  • GET /dashboard/activist/:id – Get an activist by ID (Admin only)
  • GET /dashboard/program – Get all programs (Admin only)
  • POST /dashboard/program – Enter a new program (Editor only)
  • GET /dashboard/program/:id – Get program by ID (Admin/Editor)
  • PUT /dashboard/program/:id – Edit program by ID (Admin/Editor)
  • DELETE /dashboard/program/:id – Delete program by ID (Admin/Editor)

Notes for Developers

1. All endpoints are protected; only users with correct roles can access them.
2. Use JWT access tokens in Authorization header for API requests.
3. Refresh token is stored in httpOnly cookie for token renewal.
4. Admin, Editor, and User roles are supported.
