Welcome to the MyServer Backend API

This backend powers a role-based program management platform.

Available API Endpoints:

  • GET /dashboard/activists – List all activists (Admin only)
  • GET /dashboard/activist/:id – Get an activist by ID (Admin only)
  • GET /dashboard/program – Get all programs (Admin only)
  • POST /dashboard/program – Enter a new program (Editor only)
  • GET /dashboard/program/:id – Get program by ID (Admin/Editor)
  • PUT /dashboard/program/:id – Edit program by ID (Admin/Editor)
  • DELETE /dashboard/program/:id – Delete program by ID (Admin/Editor)

To use these endpoints, you need proper authentication and roles.
